# BlockGate 启动器界面设计方案

**设计目标：完全像素化 Minecraft 启动器界面**

## 设计概述

本设计方案旨在将 BlockGate 启动器的界面完全改造为像素化 Minecraft 风格，使其与现有启动器完全不同，具有极高的辨识度和统一的游戏体验。

---

## 第一部分：整体架构与布局

### 布局结构

- **左侧可折叠侧边栏**
  - 展开宽度：200px
  - 折叠宽度：60px
  - 包含主要导航菜单

- **顶部状态栏**
  - 高度：40px
  - 显示用户信息、搜索、设置等快捷功能

- **主内容区域**
  - 占据剩余空间
  - 使用卡片网格展示游戏实例和功能模块

- **动态背景层**
  - 位于整个界面后方
  - 使用 Canvas 实现移动云和昼夜循环效果

### 导航菜单项

1. 启动（主页）
2. 游戏实例
3. 下载
4. 联机
5. 设置
6. 账户

### 响应式设计

- 窗口宽度 < 800px：侧边栏自动折叠
- 窗口宽度 < 600px：卡片网格从 3 列变为 2 列

## 第四部分：卡片网格设计

### 卡片结构

- **卡片尺寸**
  - 宽度：280px
  - 高度：200px
  - 间距：20px

- **卡片内容**
  - 顶部：游戏实例图标（80x80px，Minecraft 方块或物品图标）
  - 中间：游戏名称（像素字体，18px）
  - 底部：版本信息、最后启动时间（像素字体，14px）

### 像素化边框效果

- 边框宽度：4px
- 外边框颜色：深色（#3F3F3F）
- 内边框颜色：浅色（#BFBFBF）
- 悬停边框：草绿色外边框 + 浅绿色内边框

### 卡片背景

- **默认背景**：石头灰色半透明（rgba(127, 127, 127, 0.8)）
- **悬停背景**：泥土棕色半透明（rgba(139, 69, 19, 0.9)）
- **选中背景**：木头棕色半透明（rgba(196, 167, 108, 0.9)）

### 悬停动画效果

- 卡片放大 5%（scale(1.05)）
- 显示"启动"按钮（Minecraft 菜单按钮样式）
- 边框颜色变化（0.3s 过渡动画）

### 网格布局

- 默认：3 列布局，间距 20px
- 响应式：
  - 窗口宽度 < 1000px：2 列
  - 窗口宽度 < 600px：1 列
- 实现：CSS Grid 自适应布局

## 第五部分：按钮与交互元素设计

### Minecraft 菜单按钮样式

- **按钮尺寸**
  - 宽度：自适应内容（最小 120px）
  - 高度：40px

- **像素化边框**
  - 外边框：深色（#3F3F3F），4px
  - 内边框：浅色（#BFBFBF），2px
  - 创建立体效果，模拟 Minecraft 按钮

- **背景颜色**
  - 默认：石头灰色（#7F7F7F）
  - 悬停：泥土棕色（#8B4513）
  - 点击：深棕色（#5C3317）

- **文字样式**
  - 字体：像素字体
  - 颜色：白色
  - 对齐：居中

### 悬停效果

- 背景颜色变化（0.2s 过渡）
- 向上移动（translateY(-2px)），模拟"浮起"效果
- 边框颜色变化：外边框变亮，内边框变暗

### 点击效果

- 背景变暗，模拟"按下"效果
- 向下移动（translateY(1px)）
- 边框反转：外边框变暗，内边框变亮

### 特殊按钮类型

- **主要按钮**：草绿色背景，用于启动游戏等主要操作
- **危险按钮**：红石红色背景，用于删除等危险操作
- **禁用按钮**：深灰色背景，文字变暗，不可点击

### 输入框样式

- 像素化边框（与按钮相同）
- 背景：深色半透明（rgba(0, 0, 0, 0.5)）
- 文字：像素字体，白色
- 聚焦边框：钻石蓝色

## 第六部分：动态背景设计

### 背景实现方式

- 使用 HTML5 Canvas 在整个界面后方绘制
- Canvas 尺寸：覆盖整个窗口，响应窗口大小变化
- 渲染层级：z-index: -1，位于所有界面元素后方

### 动态元素

#### 1. 移动的云

- 云朵数量：5-8 个，随机分布
- 云朵样式：像素化白色方块组合，模拟 Minecraft 云朵
- 移动速度：缓慢向右移动（每秒 20-40px）
- 循环：移出右侧后从左侧重新出现

#### 2. 昼夜循环

- 循环周期：60 秒完成一个完整昼夜循环（可调整）
- 天空颜色变化：
  - 白天：天空蓝色（#87CEEB）
  - 黄昏：橙红色渐变（#FF6B6B 到 #FFD700）
  - 夜晚：深蓝色（#191970）
  - 黎明：粉红色渐变（#FFB6C1 到 #87CEEB）
- 星星：夜晚显示像素化星星，随机闪烁

#### 3. 地面元素

- 底部 20% 区域显示像素化地面
- 地面材质：草方块、泥土、石头等 Minecraft 方块
- 地面高度：随机变化，模拟地形起伏

### 性能优化

- 使用 requestAnimationFrame 实现流畅动画
- 降低渲染频率：每 3 帧更新一次云朵位置
- 使用离屏 Canvas 缓存静态元素（地面、星星）
- 窗口不可见时暂停动画（Page Visibility API）

### 用户控制

- 设置中提供"动态背景"开关，允许用户关闭以节省性能
- 提供"昼夜循环速度"调节（慢速/正常/快速/关闭）

## 技术实现要点

### CSS 技术栈

- 使用 CSS Grid 和 Flexbox 实现响应式布局
- 使用 CSS 变量管理颜色方案
- 使用 CSS transitions 和 animations 实现交互效果
- 使用 box-shadow 和 border 实现像素化边框

### Canvas 技术

- 使用 requestAnimationFrame 实现流畅动画
- 使用离屏 Canvas 优化性能
- 使用 Intersection Observer 检测元素可见性

### 字体加载

- 使用 Google Fonts 或本地字体文件加载像素字体
- 确保字体在所有设备上正确显示

### 图标资源

- 使用 Minecraft 官方资源或自定义像素化图标
- 所有图标保持像素化风格，尺寸统一

---

## 设计原则总结

1. **完全像素化**：所有界面元素都使用像素化风格，确保视觉统一性
2. **高度辨识度**：一眼就能看出是 Minecraft 相关应用
3. **用户体验**：保持现代应用的易用性和响应性
4. **性能优化**：动态背景等元素需要性能优化，确保流畅运行
5. **响应式设计**：适应不同屏幕尺寸，保持良好的用户体验
